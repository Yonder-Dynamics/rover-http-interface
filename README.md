# rover-http-interface

##Need:

In Chrome:

go to port (localhost:8000)

- ROS
- Node.js (npm)
- ikpy (inverse kinematics library) (python)
- WebGL-compliant browser (tested on chrome)


## Commands:

1. npm install
2. cd serv/
3. ./server_flask.py 8002
4. (In new terminal) npm start
