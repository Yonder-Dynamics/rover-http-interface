# rover-http-interface

Need:

ROS
Node.js
Inverse Kinematics library (python)
Flask (python)
WebGL (run in chrome)

In Chrome:

go to port (localhost:8000)

## Commands:

1. npm install
2. cd serv/
3. ./server_flask.py 8002
4. (In new terminal) npm start
